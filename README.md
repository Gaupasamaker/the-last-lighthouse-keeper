# The Last Lighthouse Keeper

"The Last Lighthouse Keeper" is an atmospheric, engine-agnostic signal-processing survival game. You sit at the terminal of the last lighthouse at the edge of the world, filtering transmissions, managing resources, and making impossible choices.

## Project Vision

This repository contains the First Playable (M1) codebase and specifications. Our primary focus is on atmospheric immersion through localized technical execution.

- **Status**: M1 Active (First Playable)
- **Engine**: Engine-agnostic (Node.js/HTML/CSS wrapper for portability)
- **Primary Deliverable**: Signal Interface & Main Menu Prototypes.

## Repository Structure

- `ui/`: Design specifications, implementation plans, and visual research.
- `ui_prototype/`: The functional M1 Main Menu scaffold and local wrapper.
- `assets/`: Looming ocean fogs and lighthouse beams (Placeholder enabled).
- Root Markdown docs: The "Source of Truth" for vision, risk management, and narrative.

## Getting Started (M1 Prototype)

To boot the M1 Front Door:
1. Ensure you have Node.js installed.
2. Navigate to `ui_prototype/`:
   ```bash
   cd ui_prototype
   ```
3. Run the local wrapper:
   ```bash
   node src/menu_server.js
   ```
4. Open your browser to `http://localhost:3000`.
