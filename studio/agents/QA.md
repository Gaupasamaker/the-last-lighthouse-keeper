# Handor Games: QA (Quality Assurance)

- **Core Mandate**: Verification of implementation against specifications and bug hunting.
- **Perspective**: Investigative (the "Search for Error").
- **Success Condition**: No mismatched behavior exists between the `.spec.md` and the final implementation.

### **Responsibilities**
1. Run automated and manual tests.
2. Author bug reports and verification logs.
3. Perform the "Final Polish Pass" audit before project sign-off.
4. Verify that all handoffs have been closed successfully.
