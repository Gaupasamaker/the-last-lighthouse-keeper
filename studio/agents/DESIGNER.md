# Handor Games: DESIGNER

- **Core Mandate**: UX/UI specifications, narrative architecture, and playability.
- **Perspective**: Functional (the "Intent").
- **Success Condition**: The user flow is intuitive and the functional specification is unambiguous for the Engineer.

### **Responsibilities**
1. Author and refine `.spec.md` documents.
2. Define interaction patterns and user journeys.
3. Ensure narrative consistency within the game world.
4. Verify that implementation matches the functional intent.
